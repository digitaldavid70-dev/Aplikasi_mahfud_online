Proyek ini dibangun menggunakan React.js Vite Supabase Lucide Icons.

Perbaikan & Fitur Terkini

  • Resolusi Masalah NaN: Semua perhitungan di Dashboard dan menu Mitra telah diperbaiki dengan penanganan nilai kosong (defaulting to 0) agar tidak muncul error "RpNaN".
  • Fitur Cetak Surat Jalan: Penambahan tombol print pada menu Logistik yang menghasilkan dokumen cetak profesional.
  • Integrasi Rekening Bank: Penambahan field "Informasi Rekening" pada Pengaturan yang akan otomatis muncul di bagian bawah setiap Invoice yang dicetak.
  • Data Mapping Database: Penyelarasan format data dari Supabase (snake_case) ke aplikasi (camelCase) untuk stabilitas antar modul.

Konfigurasi Database (Supabase)

Untuk mendukung fitur terbaru, pastikan schema database Anda sudah memiliki kolom berikut. Jalankan di **SQL Editor** Supabase:

-- 1. Tambah kolom bank_info ke tabel app_settings ALTER TABLE app_settings ADD COLUMN IF NOT EXISTS bank_info TEXT; -- 2. Buat tabel payments untuk laporan keuangan CREATE TABLE IF NOT EXISTS payments ( id TEXT PRIMARY KEY, date TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()), partner_id TEXT REFERENCES partners(id), amount NUMERIC DEFAULT 0, method TEXT, created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) );

Panduan Perawatan

Jika ingin melanjutkan pengembangan di masa depan:

  1. Gunakan npm run dev untuk menjalankan server lokal.
  2. Gunakan git push origin main untuk melakukan update otomatis ke Vercel.
  3. Data global dikelola di file src/contexts/AppContext.jsx.
